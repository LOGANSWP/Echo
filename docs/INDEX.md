## 📌 Agent 使用指引

1. **启动时**：读取本文件，了解文档全貌和模块分布
2. **执行任务前**：
   - 根据 `AGENTS.md` §0.2 的任务类型映射表
   - 结合本索引的“按模块快速定位”表
   - 按需读取具体文档的章节
3. **遇到疑问**：回到本索引，确认是否遗漏了相关文档
4. **任务执行中**：参考 `task-status.json` 确认当前进度和依赖
